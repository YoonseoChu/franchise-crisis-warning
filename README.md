# 소상공인 위기 조기경보 시스템 구축
---
**요약**

- 다차원 데이터와 이중 모델링 전략을 활용하여 폐업과 매출 급락을 예측한 AI 기반 소상공인 위기 조기경보 시스템 구축


**성과**

- 매출 급락 예측
    - CatBoost 모델 **QWK 0.914** 달성
    - 상위 3% 점포로 급락 이벤트 **54% 사전 탐지**
- 폐업 예측
    - K-Means 클러스터링 기반 LightGBM 2단계 모델
    - 극소 예산(3%) 투입 시 폐업 케이스 **94% 감지** $Recall@3% \approx 0.995$
    

**시기**

- 2025.09.01 ~ 2025.10.24
